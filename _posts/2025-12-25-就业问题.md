#  多云🌥

| 时间 | 中国 | 四川 |
| :---: | :---: | :---: |
| 2015 | 1655 | 84.0 |
| 2016 | 1786 | 86.1 |
| 2017 | 1723 | - |
| 2018 | 1523 | - |
| 2019 | 1465 | - |
| 2020 | 1200 | - |
| 2021 | 1062 | - |
| 2022 | 956 | - |
| 2023 | 902 | 52.9 |
| 2024 | 942 | 53.6 |


也就是说在2032之后将会面临招生困难，学校合并甚至倒闭的问题。
那么我是否有必要坚持教育这个行业还是应该早点另寻出路呢？
如果说，在考公立学校是近10年最佳的选择，那么十年后也就是我35岁之后，那时候我还能看清风向吗？
这些问题还没有解决，但是我的确需要密切关注风口了。
