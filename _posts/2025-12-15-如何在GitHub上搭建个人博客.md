# 如何在GitHub上搭建个人博客

今天想和大家分享如何使用Jekyll和GitHub Pages搭建一个完全免费的个人博客。

## 为什么选择Jekyll？

1. **完全免费** - GitHub Pages提供免费托管
2. **静态网站** - 加载速度快，安全性高
3. **版本控制** - 使用Git管理所有内容
4. **自定义域名** - 支持绑定自己的域名

## 基本原理

GitHub Pages会自动将Jekyll项目构建成静态网站，你只需要：

- 创建Markdown格式的文章
- 配置简单的YAML文件
- 选择合适的主题（或自定义）
